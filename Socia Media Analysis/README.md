## Social Media Engagement Classification Using Synthetic Data

## This project develops a machine learning pipeline to classify social media post engagement using a synthetically generated dataset. The dataset is created using the Faker library to simulate realistic social media activity, including features such as post type, number of hashtags, caption length, likes, shares, comments, views, follower count, and sentiment score. To better reflect real-world data conditions, the dataset intentionally includes inconsistencies such as missing values, duplicate records, and noisy distributions. Exploratory Data Analysis (EDA) is performed to understand feature relationships, detect outliers, and visualize engagement distribution. Additional features such as Engagement Rate, Hashtag Density, and Influence Score are engineered to improve predictive performance.

## The task is formulated as a binary classification problem, where the goal is to predict whether a post will receive High or Low engagement. A Random Forest Classifier is trained first on the raw dataset to establish a baseline performance and then on a cleaned dataset after preprocessing steps such as duplicate removal, missing value imputation, feature scaling, and categorical encoding. Model performance is evaluated using accuracy, classification reports, cross-validation scores, and confusion matrices. Feature importance analysis is also performed to identify the factors that most influence engagement prediction. This project demonstrates the complete data science workflow from synthetic data generation to model evaluation, highlighting how proper preprocessing and feature engineering can significantly improve machine learning performance.
