# Data Science Projects
This repository contains a collection of small data science and machine learning projects. The projects focus on data analysis, visualization, and predictive modeling using Python.
The goal of this repository is to explore different data science techniques, experiment with machine learning models, and practice working with real-world datasets.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Structure

Each project is stored in its own folder and typically contains:

- Jupyter notebook with the full workflow
- data preprocessing steps
- exploratory data analysis
- model training and evaluation

Projects will be added progressively as new experiments are completed.
